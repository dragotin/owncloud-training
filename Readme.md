# Usage

Download owncloud and build ownCloud docker images

```
./oc_build <owncloud-download-url>
```

Choose a OS version

```
cd owncloud_debian
```

Start or restart ownCloud docker container

```
./run.sh
```

Install and setup a list of functions for ownCloud

```
./setup.sh
```
